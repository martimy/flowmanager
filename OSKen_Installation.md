
osken-manager is removed from later releases of os-ken. To use it, you need to install an earlier release"

```bash
pip install os-ken==3.1.1
```

or (you will get version 2.3.1)

```bash
sudo apt install python3-os-ken
```

to run osken-manager without running into eventlet problems, start it with:

```bash
python3 run_osken.py
```
