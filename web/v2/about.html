About
FlowManager
Version 0.5.0  ·  OpenFlow 1.3
A RYU controller application that gives users manual control over flows in an OpenFlow network — create, modify, and delete flow entries without any coding. Ideal for learning or using OpenFlow in a lab environment.
Features
Add, modify, and delete flow entries in flow tables
Add, modify, and delete group tables and meters
Backup and restore switch tables to and from local drive
View flow tables, group tables, and meters with sortable columns
View switch port and table statistics on the dashboard
View network topology with interactive D3 visualization
Monitor live OpenFlow messages via WebSocket
Technology Stack
Vue.js 2 (Options API)
D3.js (Topology & Stats)
RYU SDN Framework
OpenFlow 1.3
WebSocket
Fetch API
